# Card of the Day

A lightweight, mobile-first "card of the day" experience that spotlights deep-cut comic trivia, a flip animation, and collection tracking.

## Highlights
- Daily card reveal with a flip animation for trivia.
- Preferences for publishers, eras, character types, and card types.
- Collection tracking (owned + wishlist) stored locally.
- Archive search with filters for publisher and era.
- Offline-ready via a simple service worker.

## Getting Started
```bash
python -m http.server 8000
```
Then open `http://localhost:8000` in your browser.
